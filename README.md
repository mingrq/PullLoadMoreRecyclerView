# 最新版本
 1.0.0
#

### 使用
#
```
在 build.gradle 中添加
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
```
dependencies {
	        implementation 'com.github.mingrq:PullLoadMoreRecyclerView:Tag'
	}
```

#### 设置反转布局是否激活 _LinerLayout布局使用_
